## Name: "vehplayer"

- **No conflicting trademark surfaced** in the categories that matter, per
  a web search (REPORTED, `docs/MARKET_AND_PRICING.md` §8). This is a web
  search, not a register search - not sufficient to rely on for filing.
- **Descriptiveness risk**: "vehplayer" reads as "vehicle player", close
  to descriptive. Descriptive marks are harder to register and harder to
  defend. Acceptable for a niche product; if filing is pursued, file the
  **logo/stylised mark** alongside the word mark, not the word mark alone
  (`docs/MARKET_AND_PRICING.md` §8).
- **Nice classes**, if filing: Class 9 (downloadable software) and Class
  42 (SaaS) - the standard pairing for an app with a web component.

## Name: "VEPLA" (retired)

Retired entirely as of session 6 (`docs/COMPETITIVE_REASSESSMENT.md`
§7.1). An unrelated Swedish B2B app (verksamhetsplatsen.se,
construction/project-management software) already ships under this name -
different Nice class, probably not a hard legal blocker, but the naming
collision plus the confusion it caused in this repo's own documentation
(two names for the same product across different files) was reason enough
to drop it rather than pursue clearance. Do not resurrect this name
without a real reason and a real clearance check.
