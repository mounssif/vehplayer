vehplayer WebRTC probe

The phone's own address on its hotspot network - shown in the vehplayer app, usually also this device's default gateway. Phone must be streaming (Start tapped) so its HTTP, WS (8787) and STUN (3478) probe servers are up. HTTP port is whatever the app's shown /go URL says - 8080 normally, 8081+ after the port-fallback path (a known-normal state after an app restart).
